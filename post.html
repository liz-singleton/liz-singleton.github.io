Finding a Path for a Nicaragua Canal

A GIS colaboration project with Lynette Murphy, Marko Patrovic, and and Agatha Grzybowski. Using skills learned in GIS II, Liz worked with her group members to create a suitability map to find the least distructive path for a canal in Nicaragua. Below you can see one of the final maps produced.
